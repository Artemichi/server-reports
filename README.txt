Nodejs сервер для автоматизации ежедевных отчетов.

Установка:
    Для корректной работы сервера папка nodejs-report-server должна находиться в дирректории rigdata рядом с папкой reports.

Настройка:
    Нажмите сочетание клавиш WIN + R и введите shell:startup, нажимите "OK".
В открывшемся окне нажмите правой клавишей "Создать" -> "Ярлык", введите путь до файла server.exe и нажмите "Далее", "Готово".

Запуск: 
    Для запуска сервера 2 раза кликните на файл server.exe

Проверка:
    После запуска сервера появится окно консоли с адресом для проверки работы сервера.
Скопируйте адрес из окна консоли и откройте его в браузере, если сервер работает в нормальном режиме, то в браузере откроется информация о статусе модулей и сервера.
